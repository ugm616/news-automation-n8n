SCRIPT PREPROCESSING
--------------------

Before sending to TTS, process the script:

1. Expand Abbreviations:
   - Dr. → Doctor
   - Mr. → Mister
   - Mrs. → Missus
   - St. → Street
   - etc. → and so forth

2. Number Formatting:
   - 25 → twenty-five
   - $1,000 → one thousand dollars
   - 2025 → twenty twenty-five
   - 3:00 PM → three PM

3. Acronyms:
   - USA → U.S.A. (with periods for better pronunciation)
   - FBI → F.B.I.
   - CEO → C.E.O.

4. Punctuation for Pauses:
   - Use commas for natural breathing points
   - Use periods for sentence breaks
   - Use ellipsis (...) for dramatic pause
   - Use em dash (—) for interruption effect

5. Emphasis (if using SSML):
   <emphasis level="strong">important point</emphasis>
   <break time="500ms"/> (for dramatic pause)

VOICE DELIVERY GUIDELINES
--------------------------

Long-Form Videos (Rumble):
- Tone: Professional, authoritative, trustworthy
- Pacing: Moderate (not too fast)
- Energy: Consistent, engaging but not overhyped
- Pauses: Natural breathing, sentence breaks
- Inflection: Professional news anchor style

Short-Form Teasers (Social Media):
- Tone: Engaging, energetic, conversational
- Pacing: Slightly faster for urgency
- Energy: High energy, compelling
- Pauses: Minimal, maintain momentum
- Inflection: More dynamic, attention-grabbing

QUALITY CONTROL
---------------

After generating audio, check:
✓ Clear pronunciation of all words
✓ Natural sentence flow and rhythm
✓ Appropriate pacing (not too fast/slow)
✓ Consistent volume throughout
✓ No robotic artifacts or glitches
✓ Proper emphasis on key words
✓ Natural pauses at punctuation

If quality issues:
- Adjust stability/similarity settings
- Rewrite problematic sentences
- Add SSML markup for pronunciation
- Try alternative voice

AUDIO POST-PROCESSING
---------------------

Optional enhancements using FFmpeg:

1. Normalize Volume:
   ffmpeg -i input.mp3 -af loudnorm output.mp3

2. Remove Silence:
   ffmpeg -i input.mp3 -af silenceremove=1:0:-50dB output.mp3

3. Add Background Music:
   ffmpeg -i voice.mp3 -i music.mp3 -filter_complex \
   "[1:a]volume=0.15[music];[0:a][music]amix=inputs=2[out]" \
   -map "[out]" output.mp3

4. Enhance Clarity:
   ffmpeg -i input.mp3 -af "equalizer=f=3000:t=h:w=200:g=3" output.mp3

TROUBLESHOOTING
---------------

Issue: Robotic voice
Fix: Increase stability, use Neural/Premium voices

Issue: Wrong pronunciation
Fix: Use SSML phonetic spelling, respell word

Issue: Unnatural pacing
Fix: Add commas/periods, adjust speaking rate

Issue: Flat delivery
Fix: Increase style setting (ElevenLabs), adjust pitch variance

Issue: Inconsistent volume
Fix: Use loudnorm filter, check audio levels
